# 3D Effects Cookbook — depth that pops off the canvas

Brand rule: **every design carries real depth — 3D effects + topic-relevant
shapes, never a flat sticker look.** This file is the copy-paste toolkit. Each
technique is pure CSS (renders in `render.py` / `animate.py`), tuned for a social
canvas that uses `vw` units (relative to the canvas width). Pair with
`reference/directions.md` (3d-depth, isometric, claymorphism, glass-aurora,
paper-craft) and the depth motifs.

---

## The 5 laws of believable 3D (read first)

1. **One light source, held everywhere.** Pick a direction (default: top-left).
   Every highlight, shadow, and extrusion obeys it. Mixed light = fake.
2. **Two shadows, not one.** A *contact* shadow (tight, dark, close = grounds the
   object) + an *ambient* shadow (large, soft, faint = the room). One shadow alone
   reads flat.
3. **Tint shadows to the background**, never pure `#000`. `color-mix(in srgb, var(--bg) 70%, #000)`.
   Pure black punches a hole; tinted shadow sits in the scene.
4. **Highlight the lit edge.** A 1–2px light inner border / top highlight on the
   side facing the light sells the volume more than the shadow does.
5. **Occlusion = depth.** Overlap layers and let near things cast onto far things.
   Parallax offset (foreground shifted more than background) reads as Z-depth.

> Quick test: if you greyscaled it, would you still read *near vs far*? If not,
> the depth is decoration, not structure — fix the shadows.

## 1. Extruded / 3D text (the headline pops)

Stacked one-pixel-stepped shadows build a solid extrusion; finish with a soft cast.

```css
.extrude{
  font-weight:800; font-size:12vw; line-height:.95; letter-spacing:-.03em;
  color:var(--text);
  text-shadow:
    .15vw .15vw 0 var(--accent), .3vw .3vw 0 var(--accent),
    .45vw .45vw 0 var(--accent), .6vw .6vw 0 var(--accent),
    .75vw .75vw 0 color-mix(in srgb,var(--accent) 70%,#000),  /* darken the deep face */
    1.4vw 1.8vw 3vw rgba(0,0,0,.4);                            /* ambient cast */
}
```

Variants: make the steps `var(--text)` for a tonal extrude; alternate two colors
for a layered-print look; add `-webkit-text-stroke:.1vw rgba(255,255,255,.25)` for
a lit top edge.

## 2. Long shadow (flat-design 3D, retro)

A single hard diagonal block of color — clean, poster-like depth.

```css
.longshadow{ color:var(--text);
  text-shadow:1vw 1vw 0 color-mix(in srgb,var(--accent) 60%,#000),
              2vw 2vw 0 color-mix(in srgb,var(--accent) 55%,#000),
              3vw 3vw 0 color-mix(in srgb,var(--accent) 50%,#000); }
```

## 3. CSS perspective & transform-3d (tilt a card into space)

Wrap in a perspective context; rotate the child. Add a matching ground shadow.

```css
.scene{ perspective:1400px; }                 /* smaller = more dramatic */
.card3d{
  transform:rotateX(8deg) rotateY(-14deg);
  transform-style:preserve-3d;
  border-radius:2.4vw; background:var(--surface);
  box-shadow:
    -2vw 3vw 0 -1vw color-mix(in srgb,var(--bg) 60%,#000),  /* side wall */
    -3vw 6vw 6vw rgba(0,0,0,.45);                            /* cast on floor */
}
.card3d::after{ /* lit top-left edge */
  content:""; position:absolute; inset:0; border-radius:inherit;
  box-shadow:inset .2vw .2vw 0 rgba(255,255,255,.25); pointer-events:none; }
```

For a true floor shadow, add a separate blurred ellipse under the object:
```css
.ground{ position:absolute; bottom:-3vw; left:10%; width:80%; height:4vw;
  background:radial-gradient(ellipse,rgba(0,0,0,.5),transparent 70%); filter:blur(1vw); }
```

## 4. Isometric (2.5D blocks & scenes)

Project onto the iso plane, then extrude the side/front faces with skews.

```css
.iso{ transform:rotateX(55deg) rotateZ(45deg); transform-style:preserve-3d; }
/* a block = top + two extruded faces */
.block{ position:relative; background:var(--accent); }
.block::before{ /* left face */ content:""; position:absolute; top:100%; left:0;
  width:100%; height:6vw; background:color-mix(in srgb,var(--accent) 70%,#000);
  transform:skewX(45deg); transform-origin:top; }
.block::after{ /* right face */ content:""; position:absolute; top:0; left:100%;
  width:6vw; height:100%; background:color-mix(in srgb,var(--accent) 55%,#000);
  transform:skewY(45deg); transform-origin:left; }
```
Lay blocks on a faint iso grid (background of skewed lines) for a built scene.

## 5. Claymorphism (puffy inflated 3D)

Big radius + an outer drop + dual inner highlights (top-left light, bottom-right shade).

```css
.clay{ border-radius:42% 58% 56% 44% / 48% 42% 58% 52%;   /* organic blob */
  background:var(--accent);
  box-shadow:
    1.2vw 1.4vw 3vw rgba(0,0,0,.28),                       /* drop */
    inset .8vw .8vw 1.2vw rgba(255,255,255,.45),           /* lit inner */
    inset -.9vw -1vw 1.4vw color-mix(in srgb,var(--accent) 60%,#000); /* shaded inner */
}
```
Use perfectly round (`border-radius:50%`) for clay buttons/bubbles.

## 6. Glassmorphism (frosted floating panel)

Blur what's behind, hairline lit border, faint top sheen — needs something behind it.

```css
.glass{ background:rgba(255,255,255,.08); backdrop-filter:blur(2vw) saturate(140%);
  -webkit-backdrop-filter:blur(2vw) saturate(140%);
  border:1px solid rgba(255,255,255,.22); border-radius:2.4vw;
  box-shadow:0 2vw 5vw rgba(0,0,0,.35), inset 0 1px 0 rgba(255,255,255,.35); }
```
Float glass over an aurora mesh / gradient orb so the blur has color to chew on.

## 7. Neumorphism (soft extruded UI)

Same-as-background surface, lifted by twin opposite shadows. Low contrast — keep
text OFF the raised chip (place labels beside, not on, faint neumorph shapes).

```css
.neu{ background:var(--surface); border-radius:3vw;
  box-shadow: 1.4vw 1.4vw 2.8vw color-mix(in srgb,var(--bg) 55%,#000),
             -1.4vw -1.4vw 2.8vw color-mix(in srgb,var(--surface) 60%,#fff); }
.neu--inset{ box-shadow:
   inset 1.2vw 1.2vw 2.4vw color-mix(in srgb,var(--bg) 55%,#000),
   inset -1.2vw -1.2vw 2.4vw color-mix(in srgb,var(--surface) 60%,#fff); }
```

## 8. 3D orb / sphere (a glowing focal mass)

Off-center radial highlight = a lit ball; add a soft glow halo + ground shadow.

```css
.orb{ width:34vw; height:34vw; border-radius:50%;
  background:
    radial-gradient(circle at 35% 30%, #fff 0%, var(--accent) 22%,
                    color-mix(in srgb,var(--accent) 55%,#000) 72%, #000 100%);
  box-shadow:0 0 8vw var(--accent), inset -2vw -2vw 6vw rgba(0,0,0,.5);
  filter:saturate(1.1); }
```
Great as the hero behind/under a headline. Animate a slow float for motion covers.

## 9. Floating object + real ground shadow

Lift the object, drop a separate blurred contact ellipse. Optional bob animation.

```css
.float{ transform:translateY(-2vw); animation:bob 4s ease-in-out infinite alternate; }
@keyframes bob{ to{ transform:translateY(-3.5vw);} }
.float-shadow{ width:60%; height:3vw; margin:1vw auto 0;
  background:radial-gradient(ellipse,rgba(0,0,0,.45),transparent 70%); filter:blur(1.2vw);
  animation:shrink 4s ease-in-out infinite alternate; }   /* shadow tightens as it rises */
@keyframes shrink{ to{ transform:scale(.82); opacity:.7;} }
```

## 10. Paper-craft layering (stacked cut depth)

Each layer a soft drop, slightly offset, to read as stacked card stock.

```css
.layer{ border-radius:1.6vw; box-shadow:0 1.2vw 2vw rgba(0,0,0,.18); }
.layer + .layer{ margin-top:-2vw; }  /* overlap so upper casts on lower */
```

## Applying 3D to ANY design (the mandate, practically)

You do not need a "3D template" to ship depth. On any post, add **at least one**:

- A **focal in 3D**: extrude the headline (#1) OR put a 3D orb/object behind it (#8/#9).
- **Layered depth** behind: gradient mesh + gradient orb + grain (already motifs)
  so the background has near/far, not a flat fill.
- **Lifted UI**: cards/badges/chips with the two-shadow rule (#3) or neumorph (#7),
  not flat rectangles.
- A **ground/contact shadow** on the hero element so it sits in space.

Then run `qa.py` — the flatness check (luminance σ) will fail a design that has no
real depth, and the critique loop scores "Depth & atmosphere." A 3D design that
still reads flat in greyscale failed the 5 laws — revisit shadows + light.

## Performance / render notes

- `backdrop-filter` (glass) and `filter:blur` are supported in the Chromium render
  path — fine. Keep blur radii in `vw` so they scale with canvas size.
- Heavy multi-layer `box-shadow` + blur on many elements can slow the render
  slightly; it still captures correctly. Reserve the big effects for the focal.
- For motion, animate transforms/opacity only (cheap, smooth) — see `motion.md`.
