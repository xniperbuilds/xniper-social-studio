# Layout Recipes — Per-Archetype Blueprints

Composition starting points. Each is a *direction*, not a rigid template — adapt
to the brief, then enforce `design-rules.md`. Pair each with a palette from
`data/palettes.json` and a font pairing from `data/fonts.json`.

---

## 1. Quote / Statement card
**Use:** quotes, hot takes, one-line value props.
- Oversized display headline as the hero (60–70% of canvas height).
- Left- or center-anchored; tight tracking, heavy weight.
- One accent word (color or italic). Giant faint quotation mark or background
  number for depth.
- Attribution small, bottom-left; handle bottom-right.
- Background: gradient mesh or luxury-dark + grain.

## 2. Tip / Listicle (single or carousel body)
**Use:** "5 ways to…", how-to steps.
- Eyebrow label (`TIP 03`) top-left, tracked uppercase.
- Short headline, then 1–3 line body — never a wall of text.
- A big index number or icon as a graphic anchor on one side.
- Carousel: same chrome, number increments, content swaps; CTA on last slide.

## 3. Carousel cover (slide 1)
**Use:** the hook that earns the swipe.
- PURE hook — a question, bold claim, or callout. No feature list.
- Biggest type in the set; a "swipe →" or "1/6" affordance bottom corner.
- Strongest depth treatment of the whole set (it competes in-feed).

## 4. Announcement / Launch / Promo
**Use:** new drop, sale, event, release.
- Eyebrow `NEW` / `LAUNCH` / `-40%` as a pill or tracked label.
- Headline = the offer in ≤6 words. Subhead = one supporting line.
- Prominent CTA button (pill, accent fill, dark text or vice versa).
- Optional product/hero visual on one side (split composition).
- Date/handle footer.

## 5. Stat / Data card
**Use:** a single impressive number.
- The number is the hero — `text-[160px]+`, heaviest weight, accent color.
- One label above, one explanatory line below. Nothing else competes.
- Subtle grid / chart motif in the background for context, low contrast.

## 6. Testimonial / Review
**Use:** social proof.
- ★★★★★ row (SVG stars, not emoji) in accent.
- Quote ≤ 3 lines, real typographic quotes.
- Avatar (real image or monogram circle) + name + role/handle.
- Glass panel over a gradient reads premium here.

## 7. Before → After / Comparison
**Use:** transformation, vs.
- Split canvas (diagonal or 50/50). Distinct treatment each side (muted "before"
  vs vivid "after").
- Clear labels; a divider line or "VS" mark in accent.

## 8. CTA / End-card (carousel close)
**Use:** the last slide.
- One job: follow / save / link in bio. One CTA, big.
- Handle + logo prominent. Re-state the value in one line.
- Arrow / "tap the link" affordance.

## 9. Thumbnail (YouTube)
**Use:** click-through.
- ≤ 4 huge words, ultra-heavy, high contrast with an outline/shadow for pop.
- Big subject/face on one side (real image), text on the other.
- One accent burst; keep bottom-right clear (duration stamp). Reads at 120px.

---

## Composition cheatsheet
- **Rule of thirds / off-center** focal point beats dead-center by default.
- **Z or F reading path** — top-left entry, CTA at the natural exit.
- **Scale contrast** — one element dramatically bigger than the rest = hierarchy.
- **Breathing room** — when in doubt, remove an element and enlarge the rest.
- **Bleed** — let a shape or the background type run off-canvas for energy.
